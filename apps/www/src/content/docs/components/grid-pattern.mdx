| Prop              | Type                                      | Default        |
|-------------------|-------------------------------------------|----------------|
| width             | `number`                                  | variant-based  |
| height            | `number`                                  | variant-based  |
| x                 | `number`                                  | `-1`           |
| y                 | `number`                                  | `-1`           |
| squares           | `Array<[number, number]>`                 | —              |
| strokeDasharray   | `string`                                  | `"0"`          |
| variant           | `"default" \| "small" \| "large"`         | `"default"`    |
| className         | `string`                                  | —              |
| ...props          | `Record<string, unknown>`                 | —              |
