## Props

| Prop            | Type     | Default                            |  
| --------------- | -------- | ---------------------------------- | 
| `pixelSize`     | `number` | `2`                                |  
| `speed`         | `number` | `0.5`                              | 
| `waveFrequency` | `number` | `5`                                | 
| `threshold`     | `number` | `0.8`                              | 
| `grainAmount`   | `number` | `0.04`                             |  
| `gridOpacity`   | `number` | `0.2`                              |  
| `cursorRadius`  | `number` | `0.5`                              | 
| `cursorPush`    | `number` | `0.5`                              |  
| `color`         | `string` | `"#00ff00"`                        |  
| `className`     | `string` | `"w-full absolute h-full inset-0"` |
