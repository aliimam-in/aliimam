## Usage

```tsx showLineNumbers
import { Book } from "@/components/ui/book"
```

```tsx showLineNumbers
<Book
  title="The Great Adventure"
  author="John Doe"
  spineText="Adventure"
  bookmark
  variant="hardcover"
  animation="hover"
>
  <p>This is some content inside the book.</p>
</Book>
```

## Props

| Prop           | Type                                                               | Default       |  
|----------------|--------------------------------------------------------------------|---------------| 
| `children`     | `React.ReactNode`                                                  | –             |  
| `title`        | `string`                                                           | –             | 
| `author`       | `string`                                                           | –             | 
| `spineText`    | `string`                                                           | –             |  
| `color`        | `string`                                                           | `#555555`     |  
| `textColor`    | `string`                                                           | `currentColor`|  
| `bookmark`     | `boolean`                                                          | `false`       |  
| `bookmarkColor`| `string`                                                           | `red`         |  
| `depth`        | `number`                                                           | `5`           |  
| `variant`      | `"default" \| "simple" \| "hardcover" \| "notebook"`               | `"default"`   | 
| `illustration` | `React.ReactNode`                                                  | –             | 
| `width`        | `number`                                                           | –             |  
| `height`       | `number`                                                           | –             |  
| `animation`    | `"hover" \| "none"`                                                | `"hover"`     |  
| `className`    | `string`                                                           | –             |  
| `...props`     | `div` props                                                        | –             |
