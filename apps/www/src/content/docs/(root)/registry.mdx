# AI Registry

The AI Registry lets you install production-ready **shadcn/ui marketing blocks** directly into your project using the shadcn CLI.

Add the registry once and then pull blocks by name — no rigid templates, just flexible, composable components you can customize and scale across marketing sites.

Add the AI registry namespace to your `components.json`.
