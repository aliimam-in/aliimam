## Installation Icons

```sh [npm]
npm install @aliimam/icons
```

## Installation Logos

```sh [npm]
npm install @aliimam/logos
```

## How to use

AI is built with ES Modules, so it's completely tree-shakable.

Each icon can be imported as a React component, which renders an inline SVG element. This way, only the icons that are imported into your project are included in the final bundle. The rest of the icons are tree-shaken away.

### Example Icons

Additional props can be passed to adjust the icon:

```tsx
import { MoonStar } from "@aliimam/icons";

// Usage
const App = () => {
  return <MoonStar size={48} />;
};

export default App;
```

import { MoonStar, Award, Cog } from "@aliimam/icons";

<div className="flex gap-3 flex-wrap">
  <div className="w-60 h-60 flex justify-center items-center mt-6 rounded-2xl p-3 bg-code">
    <MoonStar size={64} />
  </div>
  <div className="w-60 h-60 flex justify-center items-center mt-6 rounded-2xl p-3 bg-code">
    <Award className="text-blue-500" strokeWidth={0.5} size={96} />
  </div>
  <div className="w-60 h-60 flex justify-center items-center mt-6 rounded-2xl p-3 bg-code">
    <Cog strokeWidth={2} size={64} />
  </div>
</div>

### Example Logos

Additional props can be passed to adjust the icon:

```tsx
import { Apple } from "@aliimam/logos";

// Usage
const App = () => {
  return <Apple size={48} />;
};

export default App;
```

import { Apple, Nike, Vercel, Figma } from "@aliimam/logos";

<div className="flex gap-3 flex-wrap">
  <div className="w-60 h-60 flex justify-center items-center mt-6 rounded-2xl p-3 bg-code">
    <Apple size={64} />
  </div>
  <div className="w-60 h-60 flex justify-center items-center mt-6 rounded-2xl p-3 bg-code">
    <Nike className="text-red-500" size={96} />

  <div className="w-60 h-60 flex justify-center items-center mt-6 rounded-2xl p-3 bg-code">
    <Vercel strokeWidth={2} size={64} />

## Props Icons

| name          | type     | default      |
| ------------- | -------- | ------------ |
| `size`        | _number_ | 24           |
| `color`       | _string_ | currentColor |
| `strokeWidth` | _number_ | 1            |
