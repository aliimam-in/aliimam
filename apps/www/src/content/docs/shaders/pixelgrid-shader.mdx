## Props
| Prop            | Type                                                                 | Default      |
| --------------- | -------------------------------------------------------------------- | ------------ |
| `shape`         | `"wave" \| "noise" \| "ripple" \| "swirl" \| "plasma" \| "scan"`    | `"wave"`     |
| `matrix`        | `"random" \| "bayer2" \| "bayer4" \| "bayer8"`                       | `"bayer8"`   |
| `pxSize`        | `number`                                                             | `2`          |
| `amplitude`     | `number`                                                             | `0.15`       |
| `frequency`     | `number`                                                             | `0.7`        |
| `speed`         | `number`                                                             | `0.3`        |
| `rings`         | `number`                                                             | `3`          |
| `colorFg`       | `string` (HEX)                                                       | `"#E8FF00"`  |
| `cursorMode`    | `"ripple" \| "erase" \| "burn"`                                      | `"burn"`     |
| `cursorSize`    | `number`                                                             | `0.3`        |
| `cursorScale`   | `number`                                                             | `0.2`        |
| `className`     | `string`                                                             | —            |
